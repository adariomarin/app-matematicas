# App Matemáticas - Primaria Argentina 🎓

Una aplicación interactiva para que los estudiantes de primaria aprendan y practiquen matemáticas.

## 📱 Plataforma
- **React Native** - iOS y Android
- **Backend**: Firebase (base de datos en la nube)

## 🎯 Objetivo
Ayudar a estudiantes de primaria (especialmente 6to grado) a aprender:
- Divisiones
- Raíz cuadrada
- Multiplicaciones
- Sumas y restas
- Y más temas de primaria argentina

## ✨ Características Principales
- ✅ Ejercicios interactivos
- ✅ Sistema de puntuación
- ✅ Progreso guardado en la nube
- ✅ Interfaz amigable y colorida
- ✅ Desafíos por tema
- ✅ Historial de resultados

## 📦 Stack Tecnológico
- **Frontend**: React Native
- **Backend**: Firebase
- **Database**: Firestore
- **Authentication**: Firebase Auth
